Coloque aqui os arquivos de livros da Bíblia em formato JSON. Exemplo de nome: genesis.json, exodo.json, etc.
Formato sugerido:
{
  "book": "Gênesis",
  "abbrev": "gn",
  "chapters": [
    ["No princípio criou Deus os céus e a terra.", ...],
    ...
  ]
}
Cada capítulo é um array de versículos (strings).
